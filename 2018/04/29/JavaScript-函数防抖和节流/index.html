JavaScript 函数防抖和节流

发表于 2018-04-29   |   分类于 javascript   |  

摘要:
了解 JavaScript 函数防抖和节流。

假如有一个饮水机，规定按下按钮之后2秒后开始出水，有个闲得x疼的人很快的按按钮10秒，饮水机会作何反应呢？

防抖

在按下按钮和出水的这2秒内如果还有人按按钮，饮水机会重新开始计算2秒，也就是说总是以最新的操作为基准来计算时间

使用场景

防抖：频繁操作但是只需要响应最后一次即可

  • 输入框验证，只需要对最后一次输入事件作出验证即可
  • resize事件
  • …

节流：调节频率，以一定的频率去响应

  • 搜索联想
  • 响应拖拽
  • …
