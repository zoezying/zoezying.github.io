vue-cli搭建一个项目

发表于 2016-06-07   |   分类于 vue-cli搭建一个项目   |  

摘要:
用vue-cli快速搭建一个基于vue的项目

vue-cli的安装

首先要全局安装vue-cli，打开命令行输入如下命令：（默认你是安装好了node和npm）

npm install vue-cli -g

安装完成后，可以在命令行输入如下命令进行检查是否安装成功

vue -V //注意必须是要大写是的V

使用vue-cli搭建项目

在命令行输入如下命令：

vue init webpack project-name

接着它会询问你几个问题：

  1. Project name (project-name) //项目的名称，直接回车会使用括号默认的名字
  2. Project description (a vue.js project) //项目的描述，直接回车会使用括号默认的描述
  3. Author(...) //项目的作者，如果配置过GIT的作者，它会自动读取
  4. Install vue-router?(Y/N) //是否要安装vue官方的路由，输入Y表示需要，也可以直接回车表示需要，输入N表示不需要
  5. Use ESLint to lint your code?(Y/N) //是否要安装ESLint来限制你的代码错误和风格
  6. setup unit tests with Karma + Mocha? //是否需要安装单元测试工具Karma + Mocha
  7. Setup e2e tests with Nightwatch? //是否需要安装e2e来进行用户行为模拟测试

当问题回答完毕后，vue-cli就会开始安装相应的依赖项了，需要一段时间。当都安装完成后，它会提示你：

To get started: cd project-name //进入项目文件夹 npm install //在命令行输入，进行安装依赖项 npm run dev //在开发模式下运行项目
